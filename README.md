Place all the python files in the zip file into the mugen app. 
Launch mugen_ai_app.py using the powershell python command. The interface will launch. 
The graphs show, from left to right, the character name, rating in matches against AI, win rate against CPU, number of matches, and number of learnings. 
Press mute to turn off the sound. Press start and the mugen window will automatically launch and begin learning. 
The character file will then be automatically read, a models file will be created, and an expert AI for each character will be created within it.
